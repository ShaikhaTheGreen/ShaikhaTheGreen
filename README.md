### What I work on

- **Physics-informed neural networks** for wave-equation modeling and clinical acoustics — focus of my Ph.D. dissertation, *Modeling a Clinical Acoustic Information System Using Physics-Informed Machine Learning*
- **AI safety / LLM security** — detecting prompt-injection and jailbreak attempts → [`HybridGuard`](https://github.com/ShaikhaTheGreen/HybridGuard)
- **AI for cybersecurity** — phishing, network anomaly, real-time threat detection → [`AI_n_Cybersecurity`](https://github.com/ShaikhaTheGreen/AI_n_Cybersecurity)
- **Networks &amp; information security** — wireless sensor networks, cryptography, online safety

### What I teach

Machine Learning · Information Systems · Computer Security · Computer Networks · Information Privacy &amp; Data Protection · Programming (Python / C++ / Java) · Data Structures · Discrete Mathematics.

Featured course tool: [`Device-Anatomy`](https://github.com/ShaikhaTheGreen/Device-Anatomy) — interactive 3D explorer for ISC230 IT Infrastructure ([live demo](https://shaikhathegreen.github.io/Device-Anatomy/)).
